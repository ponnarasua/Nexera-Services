# Nexera Services

**Enterprise Software Development & AI Integration**

Nexera Services is a boutique software development studio offering custom software, modern web applications, and server-side AI integrations for businesses, educational institutions, and startups.

---

## 🚀 What We Build

- **Custom Software Development** — Domain-driven microservices and type-safe API layers
- **Web Development** — React / Next.js interfaces with fluid performance
- **Educational Solutions** — Student management systems, gradebooks, and analytics panels
- **Business Automation** — Workflow webhooks, CRM/ERP bridges, and background schedulers
- **Cloud Solutions** — Serverless infrastructure, Docker containerization, CI/CD pipelines
- **AI Systems** — Server-side Gemini AI integrations with secure proxy architecture

---

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, React 19, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, Supabase, PostgreSQL, Drizzle ORM
- **AI**: Google Gemini (server-side secure proxy)
- **DevOps**: Docker, Cloud Run, CI/CD pipelines

---

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Create environment file
cp .env.example .env.local
# Add your GEMINI_API_KEY to .env.local

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
